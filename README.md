# ourdecade

Our Decade前端页面，Power By [dumi](https://d.umijs.org/zh-CN)

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
